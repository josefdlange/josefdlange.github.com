How Do I Implement a Singleton in Objective-C?

HERE, YOU LAZY FOOL:

    + (id)sharedManager {
      static MyManager *sharedMyManager = nil;
      static dispatch_once_t onceToken;
      dispatch_once(&onceToken, ^{
          sharedMyManager = [[self alloc] init];
      });
      return sharedMyManager;
    }
Yes, I know there are several more ways to do it, but this is the most common use case and frankly this is all you should need. Moreover, you should avoid singletons as much as possible -- they are not a solution to everything. They are, however, a solution to some things.
